Creative Commons is on GitHub

How to find our projects and how to report bugs

As you'll see, we have a lot of repositories on GitHub, and most of them aren't software but projects we're working on a CC.

If you have a comment or suggestion, you can usually file an issue on that project. However, feel free to leave issues for anything related to Creative Commons on the main creativecommons.org issue tracker.

Filing a bug

You'll need a GitHub account to proceed.

  1. Open a new issue on GitHub
  2. Give us as much detail as you can, and if you can, please attach a screenshot. take-a-screenshot.org/ has instructions on taking screenshots.
  3. You're all set!
